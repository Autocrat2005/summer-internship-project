# Frontend

React + Vite dashboard for the geospatial intelligence MVP.

## Environment

```bash
copy .env.example .env
```

Set:

```text
VITE_API_BASE_URL=http://127.0.0.1:4000/api
```

For Vercel, set `VITE_API_BASE_URL` to the deployed Render API URL ending in `/api`.

## Commands

```bash
npm run dev
npm run build
npm run preview
```
